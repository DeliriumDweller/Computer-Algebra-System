Computer Algebra System
=================================================
 - Интерфейс
В данный момент - консольное приложение, в будущем будет реализован интерфейс на Python

 - Функционал
Программа позволяет производить вычисления с длинными числами. Реализованы действия с натуральными, целыми и рациональными числами,
а так же с полиномами.

 - Руководство
При запуске программы будет выведено сообщение о пунктах меню. Для выбора необходимо исользовать латинские буквы, с учетом регистра,
и не вводить неверные значения, так как защита от подобных случаев пока что не доработана.
При выборе раздела n для натуральных, z для целых, q для райиональных, p для полиномов будет выведен список модулей с их описанием.
Для выбора модуля используется его номер, задаваемый числом, без лишних символов.
Для возврата в предущий пункт меню или выхода из программы используется 0.

 - Представление вхожных данных
Натуральные - натуральные числа
Пример: 4, 37682, 676783987218973982719387129
Целые - целые числа с отрицательным знаком, если необходимо
Пример: 2312, -4782932
Рациональные - числитель/знаменатель
Пример: 2/6, -5632/943
Полиномы - сам полином, в качестве переменной пока что можно использовать только х, а так же свобожный член вводить последним
Пример: x^100+1, -4/3x^3+x^2-678/12x+1090132
